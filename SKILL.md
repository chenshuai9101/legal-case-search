# 法律类案检索系统

## 一、Skill概述
- 名称：legal-case-search
- 版本：v1.0
- 目标：为法律从业者提供专业的类案检索服务
- 核心价值：类案检索+报告生成，分钟级完成

## 二、核心功能

### 2.1 类案检索
- 语义检索（自然语言查询）
- 地域/法院筛选
- 时间范围筛选
- 案件类型筛选

### 2.2 案例分析
- 裁判要点提取
- 胜诉率统计
- 裁判趋势分析

### 2.3 报告生成
- 检索报告自动生成
- 符合法院提交格式
- 可直接使用

## 三、使用方式

### 类案检索
```
search_cases(query, filters, limit)
```

### 案例分析
```
analyze_case(case_id)
```

### 报告生成
```
generate_report(case_list, template)
```

## 四、实现方案

### 数据来源
- 中国裁判文书网
- 最高人民法院案例库
- 地方法院案例库

### 检索算法
- 语义相似度计算
- TF-IDF关键词匹配
- 时效性加权

## 五、验收标准

### 质量验收
- 类案召回率 ≥85%
- Top-10命中率 ≥90%
- 报告可直接提交

### 效率验收
- 单次检索 ≤2分钟
- 报告生成 ≤1分钟
